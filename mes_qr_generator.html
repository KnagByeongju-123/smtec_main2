📖 작성 요령
1️⃣ 제목/라벨 — QR 인쇄·저장 시 표시되는 이름
예) 36호기 일상점검 / A동 게시판 / SOP-품질-001
2️⃣ URL — 반드시 https://로 시작하는 정상 주소
✅ 가능: 사내 페이지, Google 시트, Google Drive 파일, YouTube, PDF, 외부 사이트
❌ 불가: 로컬 파일경로 (file://), 인트라넷 IP주소만 (192.168... 단독)
3️⃣ URL 자동 파라미터 (⭐ 핵심 기능)
URL 끝에 파라미터를 붙이면 페이지 진입과 동시에 호기 선택 + 1~3번 항목 자동 OK 처리까지 가능합니다.
📌 사용 가능한 파라미터
?ho=호기번호 → 해당 호기 자동선택
?ho=호기번호&auto=1 → 호기 자동선택 + 1~3번 자동 OK
※ auto=1은 1번 오버로드 QR 통과효과 →
  1번(오버로드)·2번(발란스)·3번(클러치브레이크) 3개 항목 일괄 OK
💡 실전 예시
예1) 36호기 점검 — 호기만 자동선택
…/P_machine_check.html?ho=36
⭐ 예2) 36호기 + 1~3번 자동 OK 처리 (가장 많이 사용)
…/P_machine_check.html?ho=36&auto=1
→ 진입 즉시: ✅ 36호기 자동선택 + ✅ 1·2·3번 항목 OK 자동처리
점검자는 4~9번 항목만 추가 QR 스캔 + 판정하면 됨
예3) 49호기 자동 OK
…/P_machine_check.html?ho=49&auto=1
🏷 호기별 QR 부착 권장
각 호기 옆에 ?ho=호기번호&auto=1 URL의 QR을 부착하면
작업자가 QR 스캔만 하면 점검페이지 진입 + 1~3번 일괄 OK까지 자동완료
💡 링크 동작 검증 (필수!)
QR 생성 후 반드시 "🔍 링크 열기" 버튼을 눌러:
① 페이지가 정상 열리는지
② 호기 자동선택이 되는지
③ auto=1 사용 시 1~3번 자동 OK 처리되는지
모두 확인 후 인쇄하세요. 인쇄·부착 후 수정 어렵습니다.
