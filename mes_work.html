🔌Python 중계서버 API›

💡 생산실적 조회 · MES 소재로트 조회는 사내 Python 서버를 통해 Oracle MES DB에 직접 접근합니다.
같은 공장 네트워크(WiFi)에 연결된 경우에만 동작합니다.

예시 주소:
• http://192.168.1.100:5100/api/mes (같은 호스트에서 Python 실행)
• /api/mes (같은 서버에서 HTML+Python 동시 서빙)
