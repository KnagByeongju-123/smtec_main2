ℹ 사용 안내
• OCR 사용법: 로드모니터 화면을 정면에서 밝은 환경으로 촬영
• 인식 값: TOTAL(녹색 큰숫자) / CH1(노랑) / CH2(분홍)
• 시간대 자동매칭: 촬영 시각 기준 ±30분 이내 가장 가까운 시간대(8:30/10:30/11:30/14:00/15:10/16:30)에 자동 입력. 다른 시간대로 변경하려면 버튼 클릭
• 점검자 자동표시: 로그인 정보(ESG_USER)에서 자동 입력
• 호기 자동 저장/복원: 호기 1회 선택 시 즉시 저장 (💾 저장됨 표시) → 다음 접속 시 자동 복원 (✓ 자동복원됨 표시)
• 호기 마스터: machine_master_tj에서 활성 호기만 로드
• 기종 자동조회: 호기 선택 시 MES → Supabase → work_order_cache 순으로 현재 작업 기종 조회
• 일일타발수: 📡 MES조회 버튼으로 가져오기
• 저장 위치: Supabase load_monitor_check 테이블
